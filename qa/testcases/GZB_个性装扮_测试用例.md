# 个性装扮

## 入口与总览

### 功能入口

- TC-GZB-001 主界面点击头像进入个性装扮
    - 【前置】已登录，处于主界面
      【步骤】1. 点击主界面左上角玩家头像 2. 在玩家信息页点击头像区域
        - 【预期】1. 打开玩家信息页，展示玩家头像、等级、称号 2. 进入个性装扮界面，标题显示"个性装扮"，默认选中当前使用的装扮
            - 【优先级】P0

- TC-GZB-002 三个页签切换
    - 【前置】已进入个性装扮界面
      【步骤】1. 点击"头像框"页签 2. 点击"称号"页签 3. 点击"气泡"页签
        - 【预期】1. 头像框列表展示，包含默认头像框(id:14000) 2. 称号列表展示，包含条件解锁和物品解锁类称号 3. 气泡列表展示，包含默认气泡(id:34000)
            - 【优先级】P0

- TC-GZB-003 加成总览弹窗
    - 【前置】已解锁至少1个带属性加成的装扮
      【步骤】1. 进入个性装扮界面 2. 点击加成总览按钮
        - 【预期】1. 加成总览按钮在所有页签均可见 2. 弹窗展示所有已解锁装扮的属性加成总和，数值与各装扮attr_value字段累加一致
            - 【优先级】P1

### 默认装扮

- TC-GZB-004 新账号默认装扮已解锁
    - 【前置】新建账号，首次进入游戏
      【步骤】1. 完成新手引导 2. 打开个性装扮界面 3. 分别查看头像框、气泡页签
        - 【预期】1. 头像框列表中"默认"(id:14000)处于已解锁状态，icon亮起，无锁标识 2. 气泡列表中"默认"(id:34000)处于已解锁状态 3. 两者均显示"当前使用"，无倒计时
            - 【优先级】P0

## 解锁
