## 环境流转

```
Dev(开发) ──→ QA(测试) ──→ Staging(预发布) ──→ Gray(灰度) ──→ Prod(正式)
   │              │              │                │              │
   └── E01服务端   └── E03版本    └── E05网络       └── E06数据    └── E02环境管理
       E04跨平台       E01服务端       E03版本           E06数据       全量验证
```
